🩺 AI Pharmacovigilance Agent

AI-powered Pharmacovigilance using n8n, Telegram, OpenRouter AI, Google Sheets and PDF Generation.

Explore Project

📖 About Project

AI Pharmacovigilance Agent is an intelligent automation system that collects adverse drug reaction reports through Telegram, analyzes them using OpenRouter AI, stores records in Google Sheets, and automatically generates professional PDF reports using n8n workflows.

⭐ Features

🤖 AI Analysis

Analyzes ADR reports using OpenRouter AI.

📲 Telegram Bot

Collects patient ADR reports instantly.

📊 Google Sheets

Automatically stores ADR data safely.

📄 PDF Reports

Creates downloadable professional reports.

⚙️ n8n Automation

Completely automated workflow.

☁️ Cloud Ready

Deployable using GitHub and n8n.

🔄 Workflow

Telegram User → n8n Workflow → OpenRouter AI → Google Sheets → PDF Generation → Final ADR Report

🛠 Tech Stack

  • n8n
  • Telegram Bot
  • OpenRouter AI
  • Google Sheets
  • GitHub Pages
  • JavaScript
